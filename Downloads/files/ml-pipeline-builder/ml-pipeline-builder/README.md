# ML Pipeline Builder v3.0

Full-featured interactive ML Pipeline Builder with:
- 🤖 Claude AI Assistant (chat, profiling, debugging, feature ideas)
- 📊 Visualizations: Confusion Matrix, ROC Curve, Learning Curves, SHAP, 3D Scatter, Residuals, Outlier Detection
- ⚡ Live model training simulation + Multi-Pipeline Runner
- 🐳 Export: Python, Jupyter Notebook, Docker, FastAPI, requirements.txt
- 🔗 Pipeline sharing via URL
- ⌨️ Keyboard shortcuts (⌘K)
- 🎯 Onboarding tour
- 📱 Mobile responsive
- Drag & drop preprocessing reorder
- Version history + diff viewer

## Quick Start
```bash
npm install && npm run dev
```
